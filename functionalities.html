System Functionalities

User Roles

  • Borrowers: Self-register, apply for loans, view status, make repayments.
  • MFI Admins: Approve/reject loans, manage borrower repayments, view MFI users.
  • Letsema Admins: Manage MFIs, MFI Admins, and access all system data.

Core APIs

Role-based APIs for registration, loan management, repayments, and MFI administration, operating under subdomain routing per tenant.
